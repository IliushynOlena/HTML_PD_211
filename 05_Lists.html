Definition List

HTML
HTML is a standardized document markup language for viewing web pages in a browser. Browsers receive an HTML document from the server using HTTP/HTTPS protocols or open it from a local disk, then interpret the code into an interface that will be displayed on the monitor screen.
CSS
CSS is a special style language used to describe the appearance of pages. The pages themselves are written in data markup languages. CSS is the core technology of the World Wide Web, along with HTML and JavaScript.
